# Instructions for Journal (`Journal.md` - Off-grid)

Append a short note about notable things from this turn with the **JOURNAL** command (it lands under today's date automatically):

```text
=== JOURNAL ===
User told me about three church friends and a cat-sitting gig. @Friends @CatSitting
```

Past days are **read-only**. Only if the user **explicitly orders** a change to a past day may you rewrite it using the **JOURNAL_EDIT** command:

```text
=== JOURNAL_EDIT 2026-06-09 ===
- [10:00] Corrected entry text. @SomeNode
```

Do **not** UPDATE `Journal.md` directly — use JOURNAL / JOURNAL_EDIT.
